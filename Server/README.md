# Server
Just as the name suggests, the directory is used to store some practical resources of Anawaert Server. Due to the fact that Anawaert Server is now mainly used for the operation of the website, as well as Anawaert Server runs on Microsoft Windows Server 2019 Datacenter, the directory mainly contains some automated scripts and utilities. The details are shown in followings:

*   `/Automation`: Automated scripts and Webhook executable.
    -   `/Webhook`: Webhook executable files.
    -   `/WebsitesUpdate`: Windows `PowerShell` scripts that are used to refresh the website by `git push` operation of remote repository.

*   `/WindowsServices/WinSW`: Windows service wrapper executable files.
